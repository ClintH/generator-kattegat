Demos

The demos give you some ideas as to how to use some of the server-side features of Kattegat.

View the source, have a tinker and keep an eye on your browser's console and your server terminal.

Storage

Demo

Put, get and remove data on the server. Keep track of data when users come and go, and survive browser or server restarts.

Realtime

Demo

Stream small events across different devices and users in realtime. Useful for ephemeral user interaction and sensor data that you don't want to accumulate.
