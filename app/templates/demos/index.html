Demos

The demos give you some ideas as to how to use some of the server-side features of Kattegat.

View the source, have a tinker and keep an eye on your browser's console and your server terminal.

Storage

Demo

Put, get and remove data on the server. This a way keeping track of data when users come and go, and for data to survive browser and server restarts.

Realtime

Demo

Exchange small bits of data across different devices and users in realtime. Unlike the Storage feature, realtime data is not stored, so it's best for streaming data - eg coming from user interaction or sensors.
