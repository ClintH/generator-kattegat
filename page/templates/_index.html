Generating pages

At the terminal, in the base directory where you previously created the server, you can generate a new page by simply running the following command, which prompts you for a page name:

$ yo kattegat:page

Or if you want to be fancy, you can tell it the page name straight away:

$ yo kattegat:page <%= pageName %>
